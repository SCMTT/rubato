# Rubato - MIDI速度调整器

一个简单易用的MIDI文件速度调整工具，支持加速、减速和原速处理。

## 功能特性

- 🎵 **速度调整**: 支持0.1x到3.0x的速度倍率调整
  - 小于1.0x: 减速播放
  - 大于1.0x: 加速播放
  - 等于1.0x: 保持原速
- 📊 **文件信息**: 显示MIDI文件的详细信息
  - 播放时长
  - 乐器数量
  - 音符总数
  - 速度变化次数
  - 调号变化次数
- 💾 **导出保存**: 将处理后的MIDI文件保存到指定位置
- 🖥️ **图形界面**: 简洁直观的GUI操作界面

## 安装要求

- Python 3.7+
- 依赖库：
  - `pretty-midi` >= 0.2.10
  - `numpy` >= 1.21.0

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

## 使用说明

1. **选择文件**: 点击"选择文件"按钮，选择需要处理的MIDI文件
2. **查看信息**: 文件加载后会显示文件的基本信息
3. **调整速度**:
   - 使用滑动条或输入框设置速度倍率（0.1 - 3.0）
4. **处理**: 点击"处理"按钮应用速度调整
5. **导出**: 点击"导出"按钮保存处理后的文件

## 项目结构

```
rubato/
```

## 技术栈

- **GUI框架**: tkinter
- **MIDI处理**: pretty-midi
